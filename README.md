# Employee Management System

A full-featured Employee Management System built with Laravel 12, Bootstrap 5, and Chart.js. Manage employees, departments, and visualize employee distribution. Includes features like search, pagination, multi-image upload, PDF export, and responsive design.

# Features

Add, edit, and delete employees.

Upload multiple images for each employee.

Pre-filled forms for editing employee details.

Delete individual employee images.

Employee status badges (Active, On Leave, Terminated).

Responsive employee table and mobile-friendly cards.

Search employees by name, email, job title, or department.

Pagination for employee listing.

Export employee list as PDF.

Department distribution pie chart with Chart.js.

Success/error toast notifications.
